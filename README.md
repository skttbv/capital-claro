# Capital Claro

Capital Claro is a bilingual compound interest calculator with recurring contributions, selectable currency, CSV export, dark/light mode, and an interactive projection chart.

## Features

- Spanish and English interface.
- EUR, USD, and GBP formatting.
- Dark and light themes with persisted preferences.
- Recurring contribution modeling.
- Interactive SVG chart with hover details.
- CSV export.
